# Sistem Rekomendasi Produk Elektronik

Aplikasi berbasis web untuk memberikan rekomendasi produk elektronik berdasarkan kategori dan budget pengguna.

## Fitur

- Filter berdasarkan kategori
- Filter berdasarkan budget
- Sorting berdasarkan rating tertinggi
- Tampilan web interaktif

## Teknologi yang Digunakan

- Python
- Streamlit
- Pandas

## Cara Menjalankan

1. Install library:
   pip install -r requirements.txt

2. Jalankan aplikasi:
   streamlit run app.py

## Struktur Folder

- app.py → program utama
- data/produk.csv → dataset produk
- utils/rekomendasi.py → logika filtering
